# multiplayer-fusion

ביצענו 2 שינויים במשחק מהשיעור:
1.  הוספנו מגן המופיע בשדה הקרב, השחקן הראשון שלוקח אותו מוגן מפגיעת לייזר.
2.  הוספנו ניקוד לכל שחקן, כך שכאשר הוא פוגע בשחקן מסוים נוסף לו ניקוד.
